# Security Policy

Talos3D does not currently offer a formal security response SLA.

## Reporting

If you believe you have found a security vulnerability:

- do not open a public issue first
- use GitHub private vulnerability reporting for this repository if it is
  enabled
- if private reporting is not available, contact the maintainer privately
  before any public disclosure

Please include:

- affected version or commit
- impact summary
- reproduction steps
- any suggested mitigation

## Expectations

- A report may receive no response or a delayed response.
- Fixes are produced only at the maintainer's discretion.
- Coordinated disclosure is appreciated, but no publication timeline is
  guaranteed.

If you need guaranteed security handling, you should not rely on this
repository without a separate written agreement.
