# Proof Point 63: Inferred Guide Lines And Protractor Workflow

**Status**: Planned  
**Date**: 2026-04-11

## Goal

Construction guides should match professional reference-line workflows rather
than acting as simple grid-plane lines.

## Target Outcome

- guide lines can be hosted by a selected face or plane
- guide direction can be inherited from a selected edge or an existing guide
- drag placement supports inference and snapping to points, handles, edges, and
  other guide references
- axis and plane locks are available through modifiers
- angular guide creation supports a protractor-style mode with visible numeric
  angle feedback and direct numeric entry
- the same guide contract is exposed through MCP so agent-authored guide
  creation is predictable
